# XENO Portfolio

A one-page portfolio for Cayden Thompson / XENO.

## Uploading to GitHub Pages

1. Open your GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and the `images` folder.
3. Commit the changes.
4. In **Settings → Pages**, publish from the `main` branch and `/ (root)`.
5. Wait about a minute, then hard-refresh the live page.

## Updating your work

Edit the project cards inside `index.html`. Replace a card link with a specific video URL whenever you choose your strongest examples.
